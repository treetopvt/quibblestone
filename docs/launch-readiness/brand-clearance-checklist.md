# Brand Clearance - Pre-Launch Gate

> **Status:** Working title is **QuibbleStone**. This is a pre-launch gate, **not** a blocker for Slice 1.
> Build under the working title; complete this checklist before any public launch, paid listing, or marketing spend.
> _None of this is legal advice. For the formal trademark steps, use a clearance service or a trademark attorney._

---

## 1. Trademark clearance

- [ ] **Identify the relevant classes** before searching:
  - **Class 9** - downloadable / recorded software (the PWA itself)
  - **Class 41** - entertainment services, providing online games
  - **Class 28** - games & playthings (only if physical goods are ever planned)
- [ ] **USPTO knockout search** at [tmsearch.uspto.gov](https://tmsearch.uspto.gov) - word mark `QuibbleStone`, then `Quibble*` wildcard, filtered to classes 9 / 41 / 28. Record live/registered hits.
- [ ] **App store + storefront sweep** - Apple App Store, Google Play, Steam, itch.io. (Trademark DBs miss unregistered-but-in-use marks; storefronts catch them.)
- [ ] **Common-law / web search** - plain Google for `"QuibbleStone"` game/app/studio to catch unregistered users.
- [ ] **Known nearby marks to keep on file** (from initial research, none currently blocking):
  - `Quibble` - price-optimization SaaS for vacation rentals (different industry/class; single word vs. compound).
  - `The Quibbler` - Harry Potter wizarding publication. The rename away from "Quibbler" distances from this; note the faint "...Stone" echo of *Philosopher's/Sorcerer's Stone* and judge if it ever feels too close in branding.
- [ ] **Decision point:** if a confusingly similar mark exists in-class -> pick an alternate from the name shortlist or consult an attorney. Keep **5 backup names** ranked.
- [ ] **(Optional, when committed)** File an **Intent-to-Use (1(b))** application to reserve the mark before public use.
- [ ] **International:** if launching outside the US, at minimum consider EU (EUIPO) - one filing covers all member states.
